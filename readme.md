# 📚 Live Dictionary

A fully functional dictionary app with overcooked UI.

Built with the holy trinity of web development: HTML, CSS, and JavaScript. No frameworks were harmed in the making of this project.

---

## 🚀 Live Demo

Try it here: [live-dictionary](https://live-dictionary.vercel.app)

---

## 🎨 Features

- Search any English word
- Suggestions that try their best to be helpful
- Pronunciation playback, assuming the internet feels generous
- Theme toggle for people who enjoy pretending dark mode fixes their life choices

---

## 🧪 Tech Stack

- HTML (structure, or whatever passes for it)
- CSS (overdesigned glassmorphism fever dream)
- JavaScript (glue, chaos, and API negotiations)

---

## 🧷 API Sources

- Dictionary data: https://dictionaryapi.dev
- Suggestions: https://datamuse.com
